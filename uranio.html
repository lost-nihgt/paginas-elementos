Terbio

Uranio

Descripción

El uranio es un elemento químico de número atómico 92. Se encuentra en la naturaleza principalmente en forma de óxido de uranio (UO2) y es uno de los elementos más pesados conocidos. El uranio es el elemento más importante de la serie de elementos radioactivos naturales conocida como uranio-torio-actinio.

Usos

El uranio se utiliza principalmente como combustible para reactores nucleares. También se usa para producir materiales radiactivos para uso médico o para producir materiales radiactivos para uso en investigaciones científicas. Otros usos incluyen productos químicos, procesos químicos, y para producir materiales de construcción.
